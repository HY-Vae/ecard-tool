# ecard-tool library
广州易林电子商务有限公司前端业务常用JavaScript函数封装

## Install
```javascript
npm install ecard-tool --save
```

## Usage
```javascript
import { Module_name } from 'ecard-tool';

Module_name.method();
```

## Example
```javascript
import { Regular } from 'ecard-tool';

let result = Regular.isEmail('liang0x4c@gmail.com');
console.log(result); // true
```
